# OXV — Site officiel

Site web de OXV (Only Xtreme Vehicle), organisateur de track days au Circuit de Haute Saintonge.

## Contenu

- `index.html` — Site complet (single-file, HTML/CSS/JS, Three.js)
- `vercel.json` — Configuration de déploiement Vercel

## Déploiement

Hébergé sur Vercel, déploiement automatique à chaque push sur `main`.

Domaine de production : **oxvehicle.fr**
